Proyectos

Video juegos

Este proyecto es una página web inspirada en el universo de Nintendo. Incluye un carrito de compras funcional y una galería de tarjetas de personajes diseñadas con Bootstrap, ofreciendo un diseño moderno, limpio y adaptable a cualquier dispositivo. Cada tarjeta cuenta con imágenes, descripciones y efectos interactivos que mejoran la experiencia del usuario. Además, la página integra animaciones, sonidos y una interfaz intuitiva que combina la estética clásica de Nintendo con un estilo web actual, demostrando cómo se pueden aplicar HTML, CSS, JavaScript y Bootstrap para crear sitios dinámicos y atractivos..

Ver demo
Clon-El Corral

La página es un clon inspirado en el sitio oficial de El Corral, desarrollada con el framework Bootstrap. Ofrece una experiencia de navegación intuitiva donde los usuarios pueden explorar el menú, visualizar los productos, personalizarlos a su gusto y agregarlos fácilmente al carrito de compras. La estructura con acordeón organiza de manera eficiente las categorías del menú, mientras que la barra de navegación permite moverse ágilmente entre las distintas secciones del sitio, brindando una experiencia fluida y atractiva para el usuario.

Ver demo
Clon-Tiktok

es un clon de la popular plataforma TikTok, con una interfaz de usuario moderna y muy similar a la original. Está diseñado para mostrar contenido multimedia como videos e imágenes, ofreciendo una experiencia de navegación fluida y atractiva entre las distintas secciones del sitio. Además, implementa una API externa para cargar automáticamente los videos, lo que permite mostrar contenido dinámico y variado sin necesidad de almacenarlo localmente. El resultado es una aplicación visualmente impactante, funcional y adaptada a dispositivos móviles.

Ver demo

App Music

Es una aplicación web responsiva para streaming de música, utilizando Bootstrap para el diseño y la integración de la API de Deezer para acceder a una vasta biblioteca musical. La aplicación tiene un diseño moderno y elegante, inspirado en plataformas como Spotify, con énfasis en la usabilidad y la experiencia del usuario...
